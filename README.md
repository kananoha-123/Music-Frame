# 🎵 音楽再生風 壁紙メーカー (Music Wallpaper Maker)

曲を検索して選び、**音楽プレイヤー風の壁紙画像**を生成・保存できる完全静的なWebアプリです。
ジャケット写真（アートワーク）、曲名、アーティスト名、曲の長さを透過テンプレートに合成し、
アートワークの色を解析して背景グラデーションを自動生成します。

- ✅ **ログイン不要・APIキー不要・サーバー不要**（GitHub Pages だけで動きます）
- ✅ 複数曲を選択して、1曲ずつプレビュー → PNG 保存（まとめて保存も可能）
- ✅ アートワークの色に合わせたグラデーション背景（色抽出ロジックを自前実装）
- ✅ 透過のまま書き出すオプションあり
- ✅ 中央ボタンを「再生 / 一時停止」で切り替え可能

---

## デモの使い方

1. 上部の検索ボックスに曲名やアーティスト名を入力して「検索」。
2. 検索結果をクリックして曲を追加（複数可）。
3. 右側のプレビューで配置を確認。「◀ 前の曲 / 次の曲 ▶」で切り替え。
4. 「この曲を PNG 保存」で1枚ずつ、「選択中すべて保存」でまとめて保存。

### オプション
- **グラデーション背景**: アートワークの色に合わせた背景を付けます（既定ON）。
- **背景なし（透過）**: 背景を入れず、透過PNGとして書き出します。

## 楽曲データについて

楽曲の検索・アートワーク・曲の長さは **[iTunes Search API](https://performance-partners.apple.com/search-api)** から取得しています。
これは **認証不要・APIキー不要・CORS対応** のため、Spotifyのようなログインやトークン発行が一切不要で、
GitHub Pages のような静的ホスティングだけで動作します。

> なぜ Spotify ではないのか？
> Spotify の検索APIは、検索だけでも必ずアクセストークンが必要で、トークンは Client Secret から発行します。
> Client Secret をブラウザ側コード（＝GitHubで公開されるコード）に置くと誰でも盗めてしまうため、
> 「ログイン不要 ＋ 完全静的 ＋ 公開」を安全に同時に満たせません。
> そこで本アプリは、同等のメタデータ（アートワーク・曲名・アーティスト名・曲の長さ）が
> キー不要で取得できる iTunes Search API を採用しています。
>
> もし将来 Spotify を使いたい場合は、Vercel / Netlify Functions などの軽量サーバー関数を1つ立て、
> トークン発行だけサーバー側で行えば、ログイン不要のまま Spotify 検索に差し替えられます。

## ローカルで動かす

`fetch` と `<canvas>` の都合上、`file://` ではなくローカルサーバー経由で開いてください。

```bash
# このフォルダで
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

---

## GitHub Pages へ公開する

1. このフォルダの中身をリポジトリの**ルート**に置いて push します。
2. リポジトリの **Settings → Pages** を開く。
3. **Build and deployment → Source** を「Deploy from a branch」にする。
4. Branch を `main`（またはお使いのブランチ）/ `(root)` に設定して Save。
5. 数十秒後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

> `.nojekyll` ファイルを同梱しているので、Jekyll の処理をスキップして
> `js/` などのフォルダがそのまま配信されます。

## ファイル構成

```
.
├── index.html                     # 画面・UI
├── js/
│   ├── app.js                     # 検索・合成・保存のメインロジック
│   └── colorthief.js              # 色抽出（Median Cut 量子化）/ 外部依存なし
├── assets/
│   ├── template_transparent_play.png   # 透過・再生ボタン版
│   └── template_transparent_pause.png  # 透過・一時停止ボタン版
├── .nojekyll
└── README.md
```

### テンプレートの座標（1536×2048 を実測）
| 要素 | 範囲 |
|---|---|
| アートワーク枠 | x 273–1261 / y 128–1152 |
| 再生バー（グレー） | x 286–1248 / y 1512–1534 |
| 経過時間 0:00 | x≈280 / y≈1577 |

`js/app.js` 冒頭の定数 `ART` / `BAR` / `TITLE` / `ARTIST` / `DURATION` / `ELAPSED` を編集すると配置を調整できます。
